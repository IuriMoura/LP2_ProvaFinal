 # Gerenciador de Estoque

Para utilizar esse aplicativo é necessario que o úsuario tenha a ultima versão do Java JDK 8 e o eclipse com o plugin e(fx)clipse instalado.

Para baixar o eclipse va no site: https://www.eclipse.org/downloads/

Para baixar o JDK 8 va no site: https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html

Após isso devemos instalar o pluguin e(fx)clipse, para isso abra o eclipse e no menu help clique em eclipse marKetplace, com o marKetplace aberto busque pelo plugin e(fx)clipse e clique em instalar.

Agora devemos configurar o eclipse, para isso clique no menu window no canto superior do eclipse, no menu window clique em Preferences, com isso ira abrir uma tela de Preferences, nela va em java -> compiler, e clique em compiler, irá abrir o compiler, nele configure o compiler compliance level em 1.8, depois va na aba java -> installed JREs nela você deve encontrar onde esta o JRE que foi instalado e selecionar ele.

Feito isso o eclipse podera rodar esse código, portante abra esse projeto com o ecliose e clique no botão play verde que fica no canto superior do eclipse, com isso o ecplise irá carregar o gerenciador de estoque e você podera uza-lo.

Usando o gerenciador de estoque, para utiliza-lo o usuário pode adicionar produtos clicando no botão adicionar, onde ira carregar outra pagina com informaçoes onde serão preenchidas os dados do produto, mas primeiro deve escolher qual o tipo de produto, podendo ser Alimento perecivel, alimento não pereciveis, liquidos, ferramentas e materias de limpeza, então o usuario pode preencher os campos vazios com as informaçoes que cada campo pede, logo depois ele deve clicar em adicionar para adcionar o produto, voltando para a tela inicial o usuario pode ir na tela buscar e pode ver que o seu produto foi adicionado e esta visivel na lista, então ele pode clicar no produto para que a suas informaçoes possasm ser mostradas, alem da tela de busca existe a tela de remover, onde o usuario digita o nome do produto que deseja remover e clica em remover, removendo assim o produto digitado.
